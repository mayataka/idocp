Detailed Description

template<typename Type, typename ImpulseType>
class idocp::hybrid_container< Type, ImpulseType >

A container that is useful to formulate the hybrid optimal control problem. This container has the standard data (with Type), data for lift stages (with Type), data for aux stages (with Type), and data for impulse stages (with ImpulseType).

Template Parameters
TypeThe type name of the standard data type.
ImpulseTypeThe type name of the impulse data type.
