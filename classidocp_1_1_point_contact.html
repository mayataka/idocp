Computes the contact Jacobian represented in the local coordinate of the contact frame multiplied by the user-defined constanct coefficient. Before calling this function, you have to update the kinematics (with respect to the position, velocity, and acceleration) of the model in pinocchio::Data.

Parameters
[in]modelPinocchio model of the robot.
[in]dataPinocchio data of the robot kinematics.
[in]coeffCoefficient multiplied to the resultant Jacobian.
[out]JacobianJacobian of the contact frame is stored in this variable. If transpose=true, size must be Robot::dimv() x 3. If transpose=false, size must be 3 x Robot::dimv()
[in]transposeA flag for transposing the Jacobian or not. If true, the Jacobian is transposed. If false, the Jacobian is not transposed, i.e., the original Jacobian is returned. Default is false.
